# Morden Labs Website & Workspace

This project has been fully migrated to a clean, self-hosted standard TypeScript/Vite setup on Google AI Studio.

## Architecture

- **Framework**: React 19 + Vite 8
- **Routing**: TanStack Router / TanStack Start
- **Styling**: Tailwind CSS v4
- **Hosting**: Self-contained local assets, standard assets pipeline (no external CDN proxies)
